The FruitLink is a complete Unity 2D puzzle game.
Basically it'll allow you to create your own puzzle game. 
It comes complete with lots of assets, animations, effects, etc...

WHAT's IN THE FRUITLINK ?
----------------------------

The fruitlink contains quite a lot of stuff. 
Everything is in the Assets folder, and in it you'll find the following folders :

- Animations : contains all the animations in the game

- Musics: all the game musics.

- Perfabs: all the game's prefabs. You just have to drag one of these in your scene and they'll be ready (effects, fruits, line, ect..)

- Scenes: a start screen (choose welcome scene in Scenes folder to start game)

- Scripts: all the game scripts. This includes data scripts, score, ui, screen resolution,... 

- Textures: all the sprites and sprite sheets used in the game. Feel free to reuse them in your own game.

WHAT AM I SUPPOSED TO DO WITH IT ?
----------------------------------

The Fruit Link includes everything you need to start your own puzzle game.
You can either start fresh and pick the scripts you need from it, or modify the existing demo levels brick by brick, adding your own sprites and scripts as you go.

USAGE
-------------------------------
When Imported project --> File --> Build Setting (or Ctrl+ Shirt + B) --> Check scenes in build exists welcome, loading,select_map, play. If not exists click button "Add Current" to add scene.

NOTE
-------------------------------
Choose loading scene in Scenes folder to start game
